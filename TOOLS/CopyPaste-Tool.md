# CopyPaste Tool

## Description
The CopyPaste Tool allows players to easily copy and paste structures within the game environment, streamlining building and design processes.

## Features
- **Easy Selection**: Use a targeting system to select structures for copying.
- **Multiple Clipboard Storage**: Store multiple copied items in separate slots for easy pasting.
- **Undo Functionality**: Supports undoing recent copy and paste actions.

## Installation
1. Download the CopyPaste Tool from the repository.
2. Place the tool in your game's tools directory.
3. Run the setup command to initiate the tool.

## Usage
- **Copying Structures**: Select the structure and use the `Copy` command.
- **Pasting Structures**: Position the cursor where you want to place the copied structure and use the `Paste` command. 

## Configuration
Customize the CopyPaste Tool settings in the `config.json` file located in the tool's directory.

## Known Issues
- Some complex structures may not copy properly.
- Performance issues with large pasted sections in low-end devices.
